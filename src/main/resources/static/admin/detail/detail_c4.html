C.不宜流通币兑换业务




高柜人员

兑换过程操作规范

1.严格按照《残损币兑换标准》无偿为客户办理残损币兑换业务，并对客户做好兑换结果的解释工作：

  1. 能辨别面额，票面剩余四分之三（含四分之三）以上，其图案，文字能按原样连接的残缺，污损人民币，金融机构应向持有人全额兑换
  2. 能辨别面额，票面剩余二分之一（含二分之一）以上，其图案，文字能按原样连接的残缺，污损人民币，金融机构应向持有人按半额兑换。纸币呈正十字形缺少四分之一的，按原面额的一半兑换
  3. 能辨别面额，票面剩余二分之一（含二分之一）以下，不予兑换

2.客户同意，柜员填单由客户签字后，柜员应在兑换后当客户面在残损币上盖章

3.如客户不同意兑换结果，柜员应向客户出具认定证明，并告知客户相关权益
